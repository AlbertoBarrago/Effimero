What Effimero is

A privacy-first, self-hosted web analytics server. It counts unique visitors with no cookies, no localStorage, and no fingerprinting: visitor identity is a daily-salted hash computed in memory and never stored, feeding only aggregate counters and HyperLogLog sketches. Privacy is a property of the algorithm, not a promise in a policy page.

Multi-tenant by design

A single instance can serve many sites while keeping them isolated. Ingest is gated by an explicit site registry and optional per-site origin allow-lists; reads are gated by per-site tokens that unlock exactly one site. See Self-hosting for the full model.
